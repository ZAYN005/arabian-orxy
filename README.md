# Arabian Orxy Engineering Consultance — Receipt PWA

## 🚀 Deploy to GitHub Pages (Free)

### Step 1 — Create GitHub Account
Go to https://github.com and sign up (free, no credit card).

### Step 2 — Create Repository
1. Click the **+** button → **New repository**
2. Name it: `arabian-orxy` (or anything you like)
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload Files
1. Click **uploading an existing file**
2. Drag and drop ALL files from this folder (keeping folder structure)
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Under **Source** → select **main** branch → **/root**
3. Click **Save**
4. Your app is live at: `https://yourusername.github.io/arabian-orxy`

## 🌐 Connect Free .com Domain (GitHub Education)

1. Go to https://education.github.com/pack
2. Find **Namecheap** → claim your free .com domain
3. In Namecheap → **Advanced DNS** → add:
   ```
   Type: A    Host: @    Value: 185.199.108.153
   Type: A    Host: @    Value: 185.199.109.153
   Type: A    Host: @    Value: 185.199.110.153
   Type: A    Host: @    Value: 185.199.111.153
   Type: CNAME  Host: www  Value: yourusername.github.io
   ```
4. In GitHub → Settings → Pages → **Custom domain** → enter your .com
5. Check **Enforce HTTPS**
6. Wait 10–30 minutes → your app is live at `arabianorxy.com` ✅
